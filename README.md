# Blackjack Trainer

App web para aprender y practicar blackjack: estrategia básica, desviaciones avanzadas y conteo de cartas Hi-Lo.

## Estado

🚧 En desarrollo activo. Fase 1 (mesa funcional) en progreso.

## Stack

- Next.js 14 + TypeScript
- Tailwind CSS v4 + shadcn/ui
- Zustand
- Vitest

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Documentación

- [PRD completo](./docs/PRD.md)
- [Guía para agentes de IA](./AGENTS.md)

## Reglas predeterminadas

Strip de Las Vegas: 6 barajas, S17, DAS, rendición tardía, blackjack paga 3:2.
